Business Review App

An officially deployed web application where users can post their business inqueries and comment on other users' posts. Browse what other users have to offer, and have the ability to create, edit, and delete previous posts or comments made by yourself. The front-end was developed using React along with some pieces of Bootstrap, while the back-end was integrated using a MongoDB database, Express.js, and Node.js. A working authentication schema was implemented using Passport.js, and RESTful routing was followed for the landing page, register page, etc. This project mimics closely to the general idea of Yelp only without the more complex rating systems and other tidbits.

3D Maze Game Solver

My first-ever individual project that I cherish nostalgically. The project is essentially a "design your own" maze game, where a user specifies a starting and ending point using ASCII characters, but also has to create their own maze primarily using letters and the pound (#) character as walls/doors. The map is then imported and read through the command line, where my C++ program utilizes the stack and queue data structures along with depth-first-search (DFS) and breadth-first-search (BFS) algorithms to determine an optimal path through the maze. The program would then translate and output each step to complete the maze if possible onto a text file within the same directory.
